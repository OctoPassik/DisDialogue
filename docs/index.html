DisDialogue

Free, open-source Discord dialog simulator. Create and test multi-character conversations using webhooks.

Download for Windows View on GitHub

Features

Multi-Character Dialogs

Simulate conversations between multiple Discord users, each with unique names and avatars.

Real Discord Webhooks

Messages are sent through real Discord webhooks and appear as genuine messages in your channel.

Custom Avatars

Use Discord User IDs to fetch real avatars automatically, or provide any image URL.

Persistent Storage

All characters, messages, and settings are saved between sessions using SQLite.

Adjustable Timing

Control the delay between messages from 1 to 60 seconds for realistic pacing.

Bilingual Interface

Full English and Russian language support built into the application.

Who Is This For?

  • Discord Bot Developers Test how your bot responds to multi-user conversations
  • Community Managers Prototype welcome messages and onboarding flows
  • Content Creators Create realistic Discord conversation screenshots for videos and articles
  • Roleplay Communities Set up scripted interactions with multiple characters
  • Server Admins Simulate moderation scenarios for training purposes
  • Demo & Tutorials Generate realistic Discord activity for guides and tutorials

Quick Start

Option 1: Windows Executable

Download from the Releases page and run — no installation needed.

Option 2: Run from Source

git clone https://github.com/OctoPassik/DisDialogue.git cd DisDialogue pip install -r requirements.txt python main.py
